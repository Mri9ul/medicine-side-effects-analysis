# Medicine Usage, Side Effects, and Habit Formation Analysis

## Overview
This project analyzes a dataset of medicines to understand:

- common treatment categories
- common side effects
- habit-forming medicines
- therapeutic classes linked to habit formation

## Dataset
The dataset contains information about medicines including:

- medicine name
- therapeutic class
- primary use
- side effects
- substitutes
- habit-forming status

## Objectives
The main goal of this project is to explore patterns in medicine usage and identify relationships between therapeutic class and habit-forming drugs.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Insights
- most medicines are not habit-forming
- certain therapeutic classes appear more frequently
- some side effects occur more often than others
- habit-forming medicines are concentrated in specific treatment areas
